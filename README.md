# Anton Shyshenko — Portfolio

**Live:** https://banderos14.github.io

## Stack
- React 18 + Vite
- CSS (no frameworks — custom design system)
- GitHub Actions for CI/CD

## Local development

```bash
npm install
npm run dev
```

## Deploy

Push to `main` branch → GitHub Actions builds & deploys automatically.

## ⚠️ DO NOT

- Rename this repository (URL will break)
- Delete this repository  
- Change GitHub username `banderos14` (URL will break)
- Disable GitHub Pages in repo settings
- Push broken code to `main` without testing locally first

## Edit content

- **Projects** → `src/components/Projects.jsx` (top of file, `const projects = [...]`)
- **Contact links** → `src/components/Contact.jsx` (top of file, `const contacts = [...]`)
- **Skills** → `src/components/Skills.jsx` (top of file, `const skillGroups = [...]`)
- **About text** → `src/components/About.jsx`
